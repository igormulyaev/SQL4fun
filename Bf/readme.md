# BF

Реализация интерпретатора Brainfuck, доказывает Тьюринг-полноту рекурсивных SQL-запросов.

Приведены примеры программ:
- реверс входной строки; 
- умножение 2 * 3 * 7 (тест вложенных циклов);
- Hello, World!

Еще пробовал запустит квайн, 1000 ячеек памяти ему хватает. За 64к итераций вывел несколько собственных строк. Полный текст ждать долго и затратно.
